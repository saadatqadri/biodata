Django Resources

Date Mon 26 May 2014 Tags django / resources

Inspired by Eddy Chan's list, I wanted to put together a list of Django resources that I used to learn the framework, and continue to reference today.

Beginner

Django Book - outdated now, needs to be rewritten for Django 1.6

Django By Example - also needs updating, but a good selection of projects to try out

Moderate

Tango with Django - full example, written in Django 1.5, easy to follow, brilliant. One of the best tutorials out there.

Effective Django Tutorial - written by an EventBrite engineer, love the section on Class-based Views

Advanced

AngularJS and Django
